## About  
MemWipe is a **lightweight memory optimization tool** designed to **free up system resources** and **enhance performance**. It allows users to manually or automatically clean up memory, clear cache, and optimize system responsiveness.  

 - **Real-time memory monitoring**
 - **One-click memory wipe**
 - **Auto-optimization settings**
 - **Customizable cleanup options**
 - **Completely free and open-source**  

---

## Features  
### **Memory Cleaning Options**  
- **Wipe DNS Cache** – Clears DNS cache to resolve domain name mapping issues.  
- **Wipe Standby Memory** – Frees up reserved memory for inactive processes.  
- **Wipe Temp Files** – Deletes temporary files to recover storage space.  
- **Wipe Set Memory** – Targets specific memory blocks for cleanup.  
- **Wipe .NET Memory** – Clears memory used by .NET applications.  

### **Auto-Optimization Settings**  
- **Wipe every X hours while idle** – Set automated cleanups when the system is not in use.  
- **Wipe when memory usage exceeds X%** – Automatically trigger memory optimization when a defined threshold is reached.  

### **System Tray & UI Enhancements**  
- **Minimalist UI** – Easy-to-use dashboard displaying **real-time memory usage**.  
- **Quick Actions from Tray Icon** – Wipe memory instantly or access settings via right-click.

## Installation  
1. **Download MemWipe** from the [latest release](https://github.com/dudetechitout/memwipe/releases).  
2. Extract the files (if applicable).  
3. Run **MemWipe.exe** — no installation required.  
4. Customize your settings as needed for automatic or manual memory optimization.  

---

## Usage  
- **Manual Cleanup**: Click the **"Wipe"** button to free up memory instantly.  
- **Auto Optimization**: Enable background cleanup under **Settings**.  
- **System Tray Access**: Right-click the **MemWipe tray icon** for quick options.
